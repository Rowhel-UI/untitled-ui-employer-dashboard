# Untitled UI Employer Dashboard

A modern employer dashboard built with React, Tailwind CSS, and Untitled UI components imported from Figma.

## Features

- 🎨 **Untitled UI Design System** - Professionally designed components from Figma
- ⚛️ **React 18** - Modern React with hooks and functional components
- 🎨 **Tailwind CSS v4** - Utility-first CSS framework
- 🧩 **shadcn/ui** - Beautiful, accessible component library
- 📊 **Recharts** - Powerful charting library for data visualization
- 🎭 **Material UI** - Additional UI components
- 🔧 **Radix UI** - Unstyled, accessible component primitives

## Tech Stack

- **Framework**: React 18.3.1
- **Styling**: Tailwind CSS 4.1.12
- **Build Tool**: Vite 6.3.5
- **Package Manager**: pnpm
- **UI Components**: 
  - Radix UI primitives
  - shadcn/ui components
  - Material UI
  - Lucide icons

## Getting Started

### Prerequisites

- Node.js 18+ 
- pnpm

### Installation

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev
```

## Figma Integration

This project includes components exported from Figma using the Untitled UI design system. The imported components maintain design fidelity while being fully customizable with React and Tailwind.
